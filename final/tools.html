About Us

In October 2025, we here at Pokehackia set out to make a website that anyone could use to find Generation 4 Pokémon hacks! Over time, our mission expanded to also include spreading awareness of publicly available tools for ROM hacking and the creation of a monthly newsletter! As long as the generation 4 Pokémon games have a dedicated community, we will remain as a beacon for new and old members of this community!

DSPRE Reloaded

Author: AdAstra-LD

The definitive DS Pokémon game editor! DSPRE Reloaded allows you to extract and edit ROM data, and includes features such as matrix editing, trainer editing, script editing, and more!

Visit Site

Pokemon DS Map Studio

Author: Trifindo

This tool allows you to make custom maps for you to import into your ROM hack. It includes features such as creating custom tilesets, changing map elevation, and editing collision data!

Visit Site

BTX Editor 2.0

Author: KazoWAR

This tool allows you to edit the btx files used for the overworld sprites. This tool is useful for those who want to add custom NPCs to their hack or add new Pokémon sprites!
